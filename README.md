# Budget-Calculator

Простое веб-приложение на Flask + SQLite для учёта доходов и расходов.

## Запуск

1. Установите зависимости:
   pip install -r requirements.txt

2. Запустите приложение:
   python app.py

3. Перейдите на http://localhost:5000/

### Функционал
* Добавление / удаление доходов и расходов
* Просмотр баланса и расходов по категориям
* Экспорт записей в JSON
* Данные сохраняются в SQLite
